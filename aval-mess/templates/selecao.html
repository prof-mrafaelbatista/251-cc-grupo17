Estruturas de Seleção em Python

Esta seção foi preparada para abordar as estruturas de seleção presentes na linguagem Python

As estruturas de seleção no Python podem ser divididas em 5 tipos: Sequencial, Seleção Simples, Seleção Composta, Encadeada Heterogênea, Encadeada Homogênea

Estrutura Sequencial

As ações são executadas de cima para baixo e da esquerda para a direita;

Geralmente, o ponto-e-vírgula ";" finaliza uma linha de ação, e como podemos ver, não é executada nenhuma escolha no codigo abaixo;

idade = int(input('Coloque sua idade');
print (idade)

Estrutura de Seleção Simples

Uma estrutura de seleção permite a escolha de um bloco de ações a ser executado quando uma condição for satisfeita.

idade = int(input("Digite a sua idade: "))
if idade > 17:
print("Pode entrar na festa!")

Já aqui, fica notória a forma que o computador só vai printar a permissão caso uma exigência seja cumprida; aqui somos apresentados ao if, básico da estrutura de seleção

Estrutura de Seleção Composta

Já a condição composta te deixa executar um bloco se a condição for verdadeira, e outro se for falsa. Usando if e else

idade = int(input("Digite a sua idade: "))
if idade > 18:
print("Pode entrar na festa!")
else:
print("É hora de voltar para sua casa")

Estrutura de Seleção Encadeada (Heterogênea)

A estrutura encadeada heterogênea é quando temos várias condições diferentes sendo testadas em sequência, cada uma com um possível resultado diferente.

Quando temos mais de dois caminhos possíveis para a lógica do programa é mais recomendado que usemos a cadeia Heterogênea. A partir daqui somo introduzidos ao elif que ocorre quando não for nem if nem ,else

idade = int(input("Digite a sua idade: "))
if idade > 17:
print("Pode entrar na festa!")
elif idade > 16 and idade < 18:
print("É hora de voltar para sua casa")
else:
print("Vou chamar seus pais.")

Estrutura de Seleção Aninhada (Homogênea)

Chamamos de estrutura homogênea (ou "aninhada") quando uma estrutura de decisão está dentro de outra.

Quando primeiro você verifica uma condição mais geral, e depois deseja aplicar refinamentos ou subcondições. Torna-se mais recomendado o uso das estruturas de seleção Homogêneas

nota = float(input("Digite sua nota: "))
if nota >= 7:
if nota >= 7 and nota <= 8:
print('Bom trabalho!')
elif nota > 8 and nota < 9:
print('Excelente nota!')
else:
print('Maravilhosa sua nota!')
else:
print('Precisa estudar um pouco mais!')

Match Cases

O match case é uma estrutura de seleção múltipla inspirada no switch case de outras linguagens (como C, Java, etc.) e permite comparar um valor específico com múltiplos padrões.

Por exemplo, nesse código observamos uma sequência de diferentes respostas que combinam com diferentes casos.

dia = 1
match dia:
case 1:
print("Dia útil!")
case 2:
print("Final de semana ou feriado!")
case _:
print(f"Valor {dia} inválido")
