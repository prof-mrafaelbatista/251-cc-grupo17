Trabalhando com Vetores e Matrizes em Python

Ao desenvolver programas, muitas vezes precisamos guardar e organizar vários dados de forma eficiente. É aí que entram os vetores e as matrizes! Eles são estruturas que nos ajudam a armazenar coleções de valores de maneira organizada e prática.

Em Python, essas estruturas são representadas por listas. Com elas, conseguimos guardar diversos valores em uma única variável, o que facilita a manipulação, a leitura e a escrita de dados. Vamos entender como funcionam essas estruturas na prática!

Vetor

Podemos imaginar um vetor como uma fila de caixas, onde cada caixa guarda um valor. Essas caixas estão organizadas uma após a outra, e cada uma tem um número de identificação (chamado de índice) que começa em 0.

Em Python, representamos um vetor usando uma lista comum, como neste exemplo:

numeros = [10, 20, 30, 40, 50]
print(numeros[0]) # Acessa o primeiro elemento
print(numeros[-1]) # Acessa o último elemento

Podemos acessar cada item individualmente usando colchetes e o índice desejado. O índice 0 acessa o primeiro valor, 1 o segundo, e assim por diante. Também podemos usar índices negativos para contar de trás para frente.

Percorrendo um vetor com for

Para trabalhar com todos os elementos de um vetor, é muito comum usar um laço for. Com ele, conseguimos percorrer item por item, o que é útil para exibir valores, fazer contas ou aplicar regras.

frutas = ["maçã", "banana", "uva"]
for fruta in frutas:
    print("Eu gosto de", fruta)

Esse exemplo mostra como podemos interagir com os elementos da lista. A cada repetição, a variável fruta assume o valor de um item do vetor frutas.

Modificando um vetor

Uma das vantagens das listas em Python é a flexibilidade. Podemos adicionar novos elementos, remover os que não queremos mais, ou até substituir valores existentes de forma muito simples.

lista = [1, 2, 3]
lista.append(4) # Adiciona o número 4 no final da lista
lista.remove(2) # Remove o número 2 da lista
lista[0] = 10 # Substitui o primeiro elemento pelo valor 10
print(lista)

Esses métodos ajudam a manter a lista sempre atualizada conforme a necessidade do programa.

Matriz

Se um vetor é uma linha de dados, uma matriz pode ser vista como uma tabela, com linhas e colunas. Imagine uma planilha do Excel — aquilo é uma matriz! Em Python, representamos esse tipo de estrutura usando listas de listas.

matriz = [
[1, 2, 3],
[4, 5, 6],
[7, 8, 9]
]
print(matriz[0][0]) # Primeiro elemento (linha 0, coluna 0)
print(matriz[1][2]) # Terceiro elemento da segunda linha

Cada elemento da matriz é acessado com dois índices: o primeiro representa a linha, e o segundo a coluna. Assim, conseguimos localizar qualquer valor dentro dessa "tabela".

Percorrendo uma matriz

Para ler todos os dados de uma matriz, usamos dois laços for: o primeiro percorre as linhas e o segundo percorre os elementos de cada linha. É como ler linha por linha de uma tabela.

for linha in matriz:
    for elemento in linha:
        print(elemento, end=" ")
    print()

A função print() no final do primeiro laço serve para pular uma linha e manter o formato de tabela na hora de imprimir.

Construindo uma matriz com input

Também podemos criar matrizes dinamicamente, pedindo ao usuário que insira os valores. Isso torna nossos programas mais interativos e personalizados.

linhas = 2
colunas = 3
matriz = []
for i in range(linhas):
    linha = []
    for j in range(colunas):
        valor = int(input(f"Digite o valor para [{i}][{j}]: "))
        linha.append(valor)
    matriz.append(linha)

No final desse código, teremos uma matriz montada com os valores que o usuário informou, prontos para serem processados pelo programa.
