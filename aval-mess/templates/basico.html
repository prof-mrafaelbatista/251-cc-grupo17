Introdução à Programação com Python

Esta seção foi cuidadosamente elaborada para introduzir você, estudante interessado em descobrir mais sobre Python. Aqui, vamos abordar apenas os conceitos básicos da linguagem, de modo que serão apresentados tipos de dados, variáveis e constantes.

Iremos abordar, junto a exemplos práticos, as partes mais essenciais para quem quer aprender sobre essa linguagem de programação tão popular.

Tipos de Dados

Os dados inseridos na programação em Python podem ser divididos em 4 diferentes tipos de natureza principais.

Tipo de Dado Primitivo Em Python Definição Exemplos
Inteiro Integer Informação numérica que pertence ao conjunto dos números inteiros relativos "1", "12", "2190"
Real Float Informação numérica que pertença ao conjunto dos números reais "2.31", "23.120", "3.14"
Caracter String Conjunto de caracteres alfanuméricos e/ou especiais. "Cachorra", "Jeitinho", "Python é Legal"
Lógico Booleano Informação composta por valores Verdadeiro ou Falso (True or False) True; False

Constante

O identificador é considerado constante quando não sofre variação ao longo da execução.

Seu valor é constante do início ao fim da execução de um algoritmo ou programa.

URL = "https://www.youtube.com/watch?v=qgffvFM1J-Q"
print(URL)

Como você pode ver, a constante URL foi declarada como tendo o valor da URL de um vídeo no Youtube. Assim, sempre que for usada, terá esse mesmo valor.

Em Python, não existe uma forma nativa de criar constantes. Por convenção, usamos nomes com letras maiúsculas e não alteramos seus valores.

Variáveis

Um dado é classificado como variável quando tem a possibilidade de ser alterado em algum instante durante a execução do algoritmo.

Exemplos de variáveis comuns:

  • Altura
  • Peso
  • Cor de Cabelo
  • Idade
  • Preços
  • Quantidade de Objetos
  • Salário

Regras para nomear variáveis: devem começar com letras (a-z, A-Z) ou underline (_) e podem ser seguidas por letras, números e underlines.

Tuplas

Tuplas são coleções de dados que são imutáveis, ou seja, uma vez criadas, seus valores não podem ser alterados.

frutas = ("maçã", "banana", "uva")
print(frutas[1]) # banana

As tuplas são úteis quando você quer garantir que os dados não serão modificados por engano.

Conjuntos (Set)

Conjuntos são coleções que não permitem valores duplicados e não garantem uma ordem específica dos elementos.

numeros = {1, 2, 3, 3, 2, 1}
print(numeros) # {1, 2, 3}

Sets são muito usados para verificar a existência de elementos e remover duplicatas automaticamente.

Dicionários

Dicionários armazenam pares de chave-valor. São úteis para representar dados estruturados.

pessoa = {"nome": "Ana", "idade": 25, "cidade": "Recife"}
print(pessoa["nome"]) # Ana

Você pode acessar valores por meio de suas chaves, além de adicionar, editar ou remover dados facilmente.
