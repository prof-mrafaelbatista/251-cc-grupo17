Funções e Procedimentos em Python

Em Python, subprogramas são blocos de código reutilizáveis que ajudam na organização, modularização e legibilidade de programas. Subprogramas podem ser funções ou procedimentos:

  • Funções: retornam um valor após a execução.
  • Procedimentos: executam uma ação, mas não retornam um valor.

Por que usar Subprogramas?

  • Reutilização: evita repetição de código.
  • Modularidade: facilita manutenção e organização.
  • Abstração: permite pensar em tarefas como unidades.
  • Colaboração: equipes podem trabalhar em funções separadas.

Funções em Python

As funções em Python são definidas com a palavra-chave def e podem ou não receber parâmetros.

def somar(a, b):
    return a + b

resultado = somar(5, 3)
print(resultado) # Saída: 8

Exemplo de Procedimento

Procedimentos são funções que não retornam valor, mas realizam ações.

def saudacao():
    print("Bem-vindo ao sistema!")

saudacao() # Saída: Bem-vindo ao sistema!

Função Principal com Condição

Python utiliza o bloco if __name__ == '__main__' para executar o código principal.

def funcao_principal():
    for i in range(1000, 2001):
        if i % 11 == 5:
            print(f"Este valor {i} -> tem resto 5")

if __name__ == '__main__':
    funcao_principal()

Essa função percorre os números de 1000 a 2000 e imprime apenas aqueles cujo resto da divisão por 11 seja 5.

A estrutura if __name__ == '__main__' garante que a função funcao_principal seja executada apenas se o script for executado diretamente (não importado como módulo).

Escopo de Variáveis

O escopo define onde uma variável pode ser acessada. Existem dois tipos:

  • Local: variável definida dentro de uma função, acessível apenas lá.
  • Global: variável definida fora das funções, acessível por todo o código.

Exemplo de Variável Local

def minha_funcao():
    mensagem = "Eu sou local!"
    print(mensagem)

minha_funcao() # Saída: Eu sou local!

Exemplo de Variável Global

mensagem = "Eu sou global!"

def minha_funcao():
    print(mensagem)

minha_funcao() # Saída: Eu sou global!

Parâmetros e Argumentos

Parâmetros são variáveis definidas na declaração da função. Argumentos são os valores passados ao chamar a função.

def apresentar(nome):
    print(f"Olá, {nome}!")

apresentar("Ana") # Saída: Olá, Ana!

Mais Exemplos de Funções

Exemplos práticos ajudam na fixação do conteúdo.

def dobro(numero):
    return numero * 2

print(dobro(7)) # Saída: 14
def verificar_par(n):
    if n % 2 == 0:
        return True
    else:
        return False

print(verificar_par(10)) # Saída: True

Conclusão

Funções e procedimentos são fundamentais na construção de programas em Python. Eles tornam o código mais organizado, reutilizável e fácil de entender. Aprender a utilizá-los corretamente é essencial para quem está começando na programação.
