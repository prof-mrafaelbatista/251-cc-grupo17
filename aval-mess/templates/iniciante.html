Para Iniciantes

Nesta seção, vamos apresentar a linguagem Python de forma clara e acessível para quem está começando agora na programação.

Python é uma linguagem muito usada e poderosa. Ela permite que você crie programas para fazer cálculos, interagir com o usuário, criar jogos, automatizar tarefas e muito mais. Vamos aprender os primeiros passos de forma prática e simples.

Vamos abordar os conceitos básicos, como valores, variáveis, comandos e estruturas. Com isso, você será capaz de entender o funcionamento da linguagem e criar seus primeiros códigos.

Conceitos Iniciais

Imagine que você quer ensinar algo ao computador. Para isso, você precisa usar uma linguagem que ele entenda. Essa linguagem é o Python.

Programar é escrever instruções que dizem ao computador o que fazer. E como ele não entende português, usamos Python para nos comunicar com ele.

print('Olá Mundo')

Esse comando faz o computador mostrar a mensagem "Olá Mundo" na tela. O "print" é usado para imprimir algo no terminal.

Para escrever seus códigos, você pode usar um programa chamado IDE (Ambiente de Desenvolvimento Integrado). Ele ajuda você a escrever, testar e organizar seus códigos.

Algumas IDEs famosas para Python são o Visual Studio Code e o PyCharm.

Comandos Básicos

Alguns comandos e tipos de dados são essenciais para começar a programar em Python:

  • print: exibe mensagens na tela.
  • #: usado para fazer comentários (o computador ignora).

Tipos de valores em Python:

  • int: números inteiros (ex: 1, 2, 3)
  • float: números decimais (ex: 3.14, 2.5)
  • str: textos (ex: "Python")
  • bool: verdadeiro ou falso (True ou False)
inteiro = 8
decimal = 3.5
texto = "Python"
logico = False

Entrada de Dados com input()

A função input() permite que o programa receba informações do usuário. Quando usamos input(), o programa para e espera o usuário digitar algo.

nome = input("Qual é o seu nome? ")
print("Olá, " + nome + "!")

Nesse exemplo, o usuário digita o nome e o computador responde com uma saudação personalizada. O que for digitado é sempre considerado um texto (str), mesmo que seja um número.

idade = input("Qual é a sua idade? ")
print("Você tem " + idade + " anos.")

Estruturas de Seleção

As estruturas de seleção permitem que o programa tome decisões com base em condições.

clima = "chovendo"
if clima == "chovendo":
print("Pegue o guarda-chuva!")
elif clima == "frio":
print("Pegue o casaco")
else:
print("Aproveite o dia!")

Estruturas de Repetição

Essas estruturas fazem com que um trecho de código se repita.

  • for: repete um número conhecido de vezes.
  • while: repete enquanto uma condição for verdadeira.
for i in range(5):
print("Bom dia!")

Listas (Vetores) e Matrizes

Listas armazenam vários valores em uma única variável. Matrizes são listas dentro de listas, organizadas em linhas e colunas.

frutas = ["maçã", "banana", "laranja"]
print(frutas[0]) # Mostra "maçã"
sala = [ ["Ana", "Bruno"], ["Carlos", "Duda"] ]
print(sala[0][1]) # Mostra "Bruno"

Funções e Procedimentos

Funções são blocos de código que realizam tarefas específicas. Procedimentos são funções que não retornam valores, apenas executam ações.

def diga_ola():
print("Olá! Tudo bem?")
diga_ola()
def somar(a, b):
print(a + b)
somar(3, 4) # Vai mostrar 7

Tratamento de Exceções

Às vezes ocorrem erros no programa. O tratamento de exceções evita que o programa pare inesperadamente.

try:
numero = int(input("Digite um número: "))
print(10 / numero)
except:
print("Erro! Número inválido ou divisão por zero.")
