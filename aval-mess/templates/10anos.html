Para Crianças Novas

Python é como brincar de montar blocos! Com ele, podemos dar instruções para o computador fazer coisas bem legais, como contar, desenhar ou até contar histórias. Vamos aprender juntos, de um jeitinho fácil e divertido.

Aqui, você vai se divertir e brincar com os valores de Python. Vamos usar palavras mágicas que dizem ao computador o que fazer. Você vai aprender o que são números, letras e comandos que ajudam a criar coisas incríveis!

Coisas Simples

Imagine que você tem um amigo, vocês podem fazer muitas coisas, podem brincar juntos e se divertir bastante!

Mas o seu amigo não fala que nem você, e para ensinar algo pra ele é bem complicado, por que ele não fala a mesma lingua que você!

Agora imagine um dia, que você achou um caderno mágico, e seu amigo entende tudo que você escreve nele! Agora vocês podem conversar.

A linguagem Python é esse caderninho e o computador é o seu amigo. Você pode brincar bastante com o computador, mas ele só vai entender oque você quer, se você usar o caderninho.

Quando você está ensinando o computador a fazer algo, você está "programando". Programar algo, é dar ordens pro seu computador fazer alguma coisa

O computador não entende tudo que a gente fala, então usamos a linguagem chamada Python, que é uma das mais fáceis e legais de aprender. Com ela, você pode: Falar com o computador, Criar joguinhos, Desenhar com códigos, Fazer o computador contar histórias e até tocar sons!

A IDE tem o papel do seu caderno, ela tem seus lápis de escrever, ela pode até ter algumas dicas para quando você esquecer como são as palavras de comando!

Os valores são uma forma de separar por tipo cada coisa. Por exemplo, A laranja é uma fruta, já o brocolis é um verdura, os dois são comidas, mas são tipos diferentes

No caso do Python, temos 4 tipo de valores para separas coisas:

  • int: Usamos para numeros inteiros como 1,2,3 (ex: Você tem 2 maçãs)
  • float: número quebrados, como 2.5, 3.14 (ex: Um balão custa 6.50 reais)
  • str: palavras entre aspas (ex: "Cachorros")
  • bool: verdadeiro ou falso, True ou False (ex: Se te perguntam se 1 + 1 = 3, você responde False)
int = 8 float = 3.5 str = "cachorro" bool = False

Estruturas de Seleção

Agora vamos falar de estruturas de seleção, que é quando um computador faz uma escolha do que fazer, dependendo da situação

Existem três tipos de seleção. "IF"': Que seria o "Se" algo acontecer, "ELSE": É o "Senão", que se o IF não estiver acontecendo, o ELSE que ocorre no lugar. Porém, também existe o "ELIF": Que seria uma escolha diferente de IF e de ELSE

frio = False chovendo = True
if chovendo: print("Pegue o guarda-chuva!")
elif chovendo = False and frio = True: print("Pegue o casaco")

Estruturas de Repetição

Agora vamos falar de estruturas de repetição, é quando o computador repete uma coisa várias vezes. É como uma música que repete o refrão!

Existem dois tipos de repetiçao. "WHILE": Enquanto algo for verdade, vai-se ficar repetindo o comando, "FOR": Só vai repetir pelo número delimitado de vezes

for i in range(5):
print("Bom dia!")

Vetores e Matrizes

Os vetores e as matrizes são usados para organizar coisas. Imagine um vetor como uma lista dos seus brinquedos, para puxar o primeiro brinquedo, você pede o brinquedo na posição 1.

Já as matrizes são como uma lista de listas, você distribui ela em linhas e colunas, pode imaginar ela como um mapa da sua sala, você senta na terceira cadeira, da quarta fileira, a forma que essa organização é distribuida é chamada de Matriz

#vetor, distribuido em lista
frutas = ["maçã", "banana", "laranja"]
print(frutas[0]) # Mostra "maçã"
#matriz, distribuida com linhas e colunas
sala = [ ["Ana", "Bruno"], ["Carlos", "Duda"]]
print(sala[0][1]) # Mostra "Bruno"

Tratando as Exceções

As vezes, podemos encontrar alguns erros quando estamos programando e nos precisamos nos preparar para quando isso acontecer. O tratamento de exceções é como um plano B para quando der errado!

Por exemplo, se eu não tenho nenhum doce e quiser dividir com meus amigos, não tem como, por que é impossível dividir zero. Logo, ao criar uma exceção, você faz um plano B para suas situações de problema.

try:
numero = int(input("Digite um número: "))
print(10 / numero)
except:
print("Opa! Não dá pra dividir por zero ou digitar letra.")

Oque está acontecendo é simples, o Try está dizendo "tente fazer isso aqui", já o except fala "Se não conseguir fazer aquilo, faça isso aqui"
