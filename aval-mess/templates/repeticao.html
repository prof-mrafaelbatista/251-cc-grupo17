Estruturas de Repetição em Python

Nesta seção, vamos conhecer as estruturas de repetição da linguagem Python, que são essenciais para automatizar tarefas, percorrer coleções de dados ou executar blocos de código diversas vezes, com base em condições específicas.

Em Python, temos algumas formas principais de repetir ações: os comandos while e for, além dos controladores break e continue, que ajudam a controlar o fluxo dessas repetições.

Estrutura de Repetição com while

A estrutura while é usada quando queremos repetir algo enquanto uma determinada condição for verdadeira. Essa é uma repetição controlada por condição — ou seja, o laço só para quando a condição deixar de ser satisfeita.

Ideal para situações em que não sabemos exatamente quantas vezes o bloco deverá se repetir.

contador = 0
while contador < 5:
    print("Contando:", contador)
    contador += 1

Estrutura de Repetição com for

A estrutura for é usada quando sabemos com antecedência quantas vezes queremos repetir um bloco de código. Muito usada para percorrer listas, strings, intervalos numéricos ou qualquer objeto iterável.

for i in range(5):
    print("Valor de i:", i)

Usando range()

A função range() é comumente usada com o for para gerar uma sequência de números. Você pode controlar o ponto de partida, o ponto final e até mesmo o passo da contagem.

for i in range(1, 11, 2):
    print(i)

Neste exemplo, a contagem começa em 1 e vai até 10, pulando de 2 em 2 (ou seja, imprimindo os números ímpares).

Usando break

O comando break serve para interromper uma repetição, mesmo que a condição ainda seja verdadeira. Muito útil quando queremos sair do laço ao atingir um determinado critério.

while True:
    texto = input("Digite algo (ou 'sair' para parar): ")
    if texto == "sair":
        break
    print("Você digitou:", texto)

Usando continue

Já o comando continue é usado para pular a iteração atual e voltar para o início do laço, continuando com a próxima repetição.

for i in range(6):
    if i == 3:
        continue
    print(i)

No exemplo acima, o número 3 não será exibido porque, quando i for igual a 3, o continue fará o programa pular para a próxima repetição sem executar o print.
